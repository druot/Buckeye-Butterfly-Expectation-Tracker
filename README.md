# Buckeye Butterfly Expectations Tracker

A classroom expectations tracker where a student earns 10 points toward break time. Each time the student reaches 10 points, a "Break time!" popup appears and the tracker can start a new break cycle.

## Files

- `index.html` - app layout
- `styles.css` - visual design, responsive layout, and animations
- `script.js` - progress tracking, lifecycle stages, local saving, and reward popup

## Run Locally

Open `index.html` in a browser, or serve the folder with any static web server.

## Deploy To Vercel

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md`.
3. Go to Vercel and choose **Add New Project**.
4. Import the GitHub repository.
5. Use the default Vercel settings.
6. Deploy.

No build command is needed because this is a static app.
